General Ubiquiti Networks Setup

Network Schematic

The schematic below is an example of how all the Ubiquiti devices are connected together for a small business. To get easy to follow the schematic, it has been colored code to reflect a physical ethernet connection among devices. The orange lines are the 100 Mbps ethernet connections, the blue ones are 1 Gbps ethernet connections, the green one is 10 Gbps ethernet connection, and the red ones are powered cords.

Unifi Access Points

The setup has a Modem to get connected to the Internet, one Unifi Security Gateway, one Unifi PoE Switch 48-Port, several Unifi AC Access Point, several Unifi AC In-Wall Access Point. The Unifi AC In-Wall Access Point is the Access Point that can be mounted in a wall, where the Unifi AC Access Points cannot be mounted.

Power Over Ethernet (PoE) Cameras

Besides the Access Points, the setup also has several PoE Cameras, which are made by Ubiquiti. These cameras are power by PoE, which can be mounted almost anywhere in a building.

Server Computer

Moreover, the setup has a server computer, which is called Kakaroto Data Center, to host a web server, storage, and Network Video Recording system. The operating system of Kakaroto Data Center is unRaid, which a Unix-like system. unRaid’s user interface is very friendly and easy to use. The unRaid operating system is designed to host multiple virtual machines. Below is the specifications of the Kakaroto Data Center in the setup.

Unifi Cloud Key

The setup also has the Unifi Cloud Key to run a software defined network to control all of the Ubiquiti devices.

Battery Backup

Networking devices are known to be sensitive with power surge and power outage, to make sure the networking devices are still running on an outage event, these devices are connected to a battery backup which allows all the networking devices continue running during the power outage so that an administrator has enough time to shut down them properly.

In Conclusion

This is a general Ubiquiti Networks and scalable for a small business infrastructure setup. In other words, as the business grows, more networking devices such as switches, access points and cameras can be added to the existing system without reconfiguring anything.
