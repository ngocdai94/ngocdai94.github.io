Amazon Web Services (AWS) - A Cloud Computing Service

Up to now, you probably know a lot about Ubiquiti Enterprise Network products from Ubiquiti Unifi Access Point & Switches, Unifi Security Gateway and How to Set Up?, unRaid Operating System & Virtual Machines, and Dockers blog posts. You also know what software is usually used with those Enterprise networking products.

As I mention before, Enterprise networking products are usually very expensive. To have a similar system, it usually costs at least $10, 000 USD to build. Imagine if you are a business owner, who is planning to open a small start-up business and planning to have a powerful Database server to store data and employees’ information, you will probably look for other solutions to build your start-up business due to the initial cost for those enterprise networking products.

Fortunately, to solve this problem, a lot of companies like Microsoft, Google, and Amazon are offering Cloud Services, which helps small startups to set up a strong back-end network infrastructure for their startup. All they need is to have a good Internet connection to those Cloud Services.

What is Amazon Web Services (AWS)?

Microsoft has Microsoft Azure as its Cloud Computing Services. Google has Google Cloud Platform as its, and Amazon has a cloud computing service called Amazon Web Services. All these cloud services offer real back-end services like Web Hosting servers, virtualization, and database services like MySQL, MongoDB, etc.

These cloud services offer similar solutions to all businesses’ needs. However, as up to date, 2019, Amazon Web Services is the most dominating Cloud Services in the world.

Why Amazon Web Services is so strong?

  1. Functionalities

    2. AWS has significantly more services, and more features within those services, than any other cloud provider. The AWS cloud platform offers over 165 fully-featured services, including over 40 services that aren’t offered by any other major cloud provider. AWS provides services for a broad range of applications including compute, storage, databases, networking, analytics, machine learning and artificial intelligence (AI), Internet of Things (IoT), security, and application development, deployment, and management.

  2. Security

    3. Security is one of the main concerns nowadays. What built by humans can also be cracked by humans. Therefore, no matter how powerful a machine is configured, it is always has a risk to be attacked if an administrator does not pay attention to software updates, and the Internet packets coming in/out from the Internet Gateway. However, with hosting a whole back-end system on the cloud, users can worry-free about those security risks since AWS will constantly update and monitor those packets to prevent hackers. Furthermore, each connection to AWS is a Virtual Private Network (VPN) connection. In other words, a connection between client and AWS is encrypted.

  3. AWS Regions

    4. AWS has the most data centers around the world. This means that wherever clients go, they can always get accessed to their data instantly since the data is synced across AWS data centers.

    So far AWS has 69 Availability Zones (AZs) within 22 geographic regions around the world and announced plans for 13 more AZs and four more AWS Regions in Indonesia, Italy, South Africa, and Spain. Figure 1 shows what AWS Regions looks like up to now.



Why AWS is more successful than Microsoft Cloud Azure, and Google Cloud Computing?

Google Cloud Services and Microsoft Cloud Services have been available before Amazon Web Services. Google Cloud Services and Microsoft Cloud Services were most offered data solutions like Google Drive and OneDrive, respectively. These services aimed mostly to help clients and businesses to have dynamic storage on the cloud wherever they go.

However, when Amazon started, they started with virtualization on the cloud. In other words, computers on the cloud that can be accessed anywhere in the world. When Amazon first started this solution, it helped a lot of start-up businesses saving a lot of money to invest in strong back-end infrastructure. Instead of investing in those expensive enterprise networking products, clients can use the money to hire more employees to push more productivities. For this reason, a lot of clients back then leveraged AWS to start their businesses.

When Google and Microsoft see a huge increasing revenue in those cloud services that Amazon is offering, they start to do the same. Google is now having more services on the cloud such as Google Document, Google Excels, Google Drive, etc. Microsoft has been transitioning all its services such as Microsoft Office, Databases, etc. to the cloud. Up to now, Microsoft is a rival to Amazon Web Services. They constantly check each other services every day to ensure that both are offering the same services and competitive prices.

One of the most successful cloud services from Microsoft is Office 365, in which users can edit documents directly on the cloud, similar to Google Documents. Microsoft is trying to bring its new service called Windows 365 to the cloud that can help users use their own Windows machine on the cloud without buying a very expensive computer.

Windows 365 is a great idea and a great move by Microsoft. However, I think that it is quite scary to have your own Windows PC on the cloud. What happened if your cloud computer is hacked? Or What if a government wants to look at your data on the cloud? For these reasons, I do not think it is going to be a great service option for everyone.

What are the benefits of using Amazon AWS?

Amazon is not only the biggest Online Retailer, where you can find all the things you need on Amazon.com but also, it is the biggest Cloud Services in the world. Amazon data center is expanding, and it is improving and offering more services. When using Amazon as your services, you will have the following benefits.

  1. Extremely fast uploading/downloading to the AWS services anywhere in the world.
  2. Affordable price. Pay as you go and only pay to the services that you are using.
  3. Strong and up-to-date back-end security server.
  4. Back-end enterprise networking products worry-free.
  5. AWS is a scalable service. No matter how big your businesses are going to be, AWS will be always scalable to your businesses’ size.

Conclusion

In the next few years, cloud computing is going to be the main service. Microsoft and Apple are transitioning their software licenses to software as a service. In other words, most of the software products such as Microsoft Office 365, Microsoft Windows 365, and Windows MySQL Community Server are going to be on the cloud. To use them, users must subscribe to those services either monthly or annually. Lifetime purchase is going to disappear since there is a lot of work to maintain data center such as replacing failure disks, cooling the data center, electricity bills, etc.

Furthermore, we are going to expect the same thing to hardware as well. For example, Apple has a program called “New iPhone Ever Year.” This means that users pay Apple a monthly fee to rent an iPhone. When a new iPhone comes out, they can trade the old one to get a new one.

Another example is Google Stadia and Nvidia GeForce Now. Google Stadia and Nvidia GeForce Now are cloud gaming services. Instead of building a very expensive gaming PC to play heavy graphic games with a high frame rate per second (FPS), they can play those games directly on the cloud.

All these services always rely heavily on the Data Center. For this reason, subscription to use a service is going to be very popular in the future, and make sure you have a reserved big pocket when that time comes :).
