Virtual Machines on unRaid Operating Systems

What is Virtual Machine?

Virtual Machine is basically an operating system that is running on another host operating systems. It borrows resources on the host systems to run. The host machines can be any operating systems such as Ubuntu, unRaid, RedHat, Windows, etc. As long as the host machine’s CPU is powerful and supports Virtualization or Hyper-V, it can host a virtual machine.

Benefits of Using Virtual Machines

Virtual machines have a lot of benefits for educations and technology businesses. For educations, to teach students how to use an operating system, school IT Administrators tend to install a virtual machine on a physical desktop computer at the beginning of every quarter. The reason is that students will make the operating system’s files corrupted during the process of learning how to use a new operating system. Therefore, to prevent it the IT Administrators do not want to spend a lot of time fixing the host computers if they have file corrupted. However, they rather want to delete the virtual machines and install fresh ones.

For technology companies such as Apple, Google, and Amazon, the virtual machines will allow software developers and software engineers to test and validate their new versions of a software before releasing to consumers.

Typical Specifications of a Host Machine

  1. CPU
    • 6 Cores 12 Threads by either Intel or AMD
  2. Memory
    • 32 Gigabyte (GB) of RAM
  3. Storage
    • Main: 512 GB of Solid State Drive (SSD)
    • Data Storage: 1 Terabyte (1TB = 1,000 GB)

How To Create a Virtual Machine on unRaid?

First, an image disk is required to install. For the example, Windows 10 ISO image disk will be used. Next, navigate to a “VMS” tab on the main unRaid’s home page.

Then, choose “ADD VM” option below.

Next, choose the virtual machine to install.

When prompted, specify the location of the Windows ISO image file in the “OS Install ISO” path. The rest can leave by default. Then, click create and start the virtual machine installation process.

The Windows 10 will appear as shown after the installation is completed.

In Conclusion

As the world is moving to Cloud Computing and Internet of Things (IoT), virtual machines have been using as a firewall and internet traffic monitor to track and prevent hackers stealing users’ information.

The best practice when visiting a strange link is opening that link in a virtual machine so that bad guys on the Internet cannot get important files if the virtual machine is hacked.
