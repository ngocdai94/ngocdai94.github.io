unRaid Operating System (OS) - A New Unix-Like System

unRaid Operating System (OS)

From the previous post, I talked about how a general Ubiquiti Networks is setup, which is the combination of Unifi Cloud Key, Unifi Security Gateway, Unifi PoE Switches, Unifi Access Points, Unifi Cameras, and a Server Computer. The server computer is powered by unRaid Operating System, which is getting a lot attention recently.

unRaid Operating System is a unix-like operating system. In other words, it means that the unRaid OS is created based on a Unix system. Then, developers and software engineers add and remove some feature on the Unix to create a better Unix version. Like other popular Unix systems such as Ubuntu, Raspbian, and RedHat, unRaid OS is fast and reliable due to the operating system size. unRaid OS is specialized in Network Attached Storage (NAS) and virtual machines environment.

Versions

One thing that set unRaid OS aside is that users have to purchase a license to run unRaid OS. However, the license is a lifetime fee. There are three versions of unRaid OS licenses, which are Basic, Plus, and Pro. unRaid also comes with a 30-Day Free Trial with the Pro version for users to try it.

  1. Basic is $59
    • Up to 6 attached storage devices
  2. Plus is $89
    • Up to 12 attached storage devices
  3. Pro is $129
    • Unlimited attached storage devices

(for more information about pricing, please visit unRaid Pricing)

How To Install unRaid OS?

To install unRaid OS, a minimum 16 Gigabyte of USB storage is required. However, to ensure unRaid OS not running out of storage, 32 Gigabyte of USB storage is recommended. unRaid OS is flexible and can be transferred to any host machine by plugging the USB to it.

The unRaid team makes the installation process easier than the past. Users can visit the unRaid Download page to download a software, USB Creator, to create a bootable USB. Then, choosing the boot order on a host machine to boot to the USB and finishing the installation process.
